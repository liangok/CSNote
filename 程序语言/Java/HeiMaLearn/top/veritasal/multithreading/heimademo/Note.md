#### 单例设计模式

保证类在内存中之有一个对象

如何保证

1. 控制类的创建，不让其他类创建本类对象。private
2. 在本类中定义一个本类的对象。Singleton s；
3. 提供公共的访问模式。public static Singleton getInstance(){return s}
