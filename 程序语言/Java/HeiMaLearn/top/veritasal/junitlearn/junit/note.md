## junit单元测试
    测试分类
        1. 黑盒测试
        2. 白盒测试
        
    junit使用：百盒
        步骤：
            1.定义一个测试类
                建议：
                    - 测试类名：被测试类名 + Test
                    - 包名：xxx.xxx.xxx
            2. 定义测试方法：可独立运行
                建议：
                    方法名：test+ 测试方法名 eg: testadd()
                    返回值：void
                    参数列表：空参
                    
            3. 给方法加@Test
            4. 导入junit依赖环境
            
        结果判定：
            红色失败，绿色成功
            一般使用断言处理结果
                
        补充
            @Before：
                修饰的方法会在测试方法执行前被自动执行
            @After
                修饰的方法会在测试方法执行后被执行
